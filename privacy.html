Политика конфиденциальности

Дата обновления: 25.03.2026

Настоящая политика описывает порядок обработки персональных данных пользователей сайта VELIGODSKY.PARFUMS.

1. Какие данные мы обрабатываем

  • Контактные данные, которые вы передаете при оформлении заказа в Telegram.
  • Технические данные: IP-адрес, тип устройства, данные браузера, события на сайте.
  • Данные корзины и выбранных товаров для оформления заказа.

2. Для чего используются данные

  • Для обработки и сопровождения заказа.
  • Для связи с вами по вопросам заказа и доставки.
  • Для улучшения работы сайта и качества сервиса.

3. Передача данных третьим лицам

Данные могут передаваться только в объеме, необходимом для выполнения заказа и требований законодательства.

4. Сроки хранения

Данные хранятся не дольше, чем это требуется для целей обработки и исполнения обязательств.

5. Обращения по персональным данным

По вопросам обработки персональных данных напишите в Telegram: @veligodsky_ls.
